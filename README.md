
# Premier Metrics

Player statistic comparison and visualization tool. 


![Logo](https://premier-metrics.vercel.app/images/home/hero.jpg)


## Demo Link

[Premier Metrics Website](https://premier-metrics.vercel.app/)


## Acknowledgements
All statistical data, team logos, and player names are sourced from FBref and the official Premier League website. All the team logos and other material used belong to their respective owners. This project is strictly for educaitonal and non-commercial use.
 - [FBREF's Website](https://fbref.com/en/)
 



## Features

- Head-to-Head Comparison between Players
- Radar Graphs for visualizing how a player stacks up compared to others of the same position.
- Player Search


## Tech Stack

**Client:** React, Tailwind CSS, Chart.js (graphs).

**Back-End:** Vercel (hosting + API), JavaScript, PostgreSQL, Supabase.

**Data Pipeline**: Python (with Pandas), automated with GitHub Workflows.
